# Chapter 3: ROS 2 - The Robotic Nervous System

**Weeks 3-4 | Module 1: ROS 2 Foundations**

## Learning Objectives

By the end of this chapter, you will be able to:

- Install and configure ROS 2 Humble on Ubuntu 22.04
- Create ROS 2 workspaces and packages
- Implement publisher and subscriber nodes in Python
- Use ROS 2 command-line tools for debugging
- Understand the ROS 2 computation graph architecture

## Chapter Contents

1. [ROS 2 Installation](./installation) - Setting up ROS 2 Humble
2. [Core Concepts](./core-concepts) - Nodes, topics, messages
3. [Your First Publisher](./first-publisher) - Sending messages
4. [Your First Subscriber](./first-subscriber) - Receiving messages
5. [ROS 2 CLI Tools](./cli-tools) - Command-line debugging

## Key Concepts

| Concept | Description |
|---------|-------------|
| Node | Executable that performs computation in ROS 2 |
| Topic | Named bus for nodes to exchange messages |
| Message | Typed data structure for communication |
| Publisher | Node that sends messages to a topic |
| Subscriber | Node that receives messages from a topic |

## Prerequisites

- Ubuntu 22.04 (or WSL2 on Windows)
- Python 3.10+
- Chapter 2 completed

**12-15 hours** of study and hands-on exercises
