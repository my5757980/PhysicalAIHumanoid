## Module 1: ROS 2 Foundations (Weeks 1-5)

### Chapter 1: Introduction to Physical AI
**You are here!**

- What is Physical AI?
- History of humanoid robotics
- Simulation-first approach
- Development environment setup

**Key Outcome**: Understand the field and have a working dev environment

### Chapter 2: Foundations of Humanoid Robotics

- Humanoid anatomy and degrees of freedom
- Forward and inverse kinematics
- Sensor technologies
- Actuator types

**Key Outcome**: Calculate kinematics and understand robot hardware

### Chapter 3: ROS 2 Basics

- ROS 2 architecture and concepts
- Nodes, topics, and messages
- Publishers and subscribers
- CLI tools and debugging

**Key Outcome**: Build basic ROS 2 applications

### Chapter 4: ROS 2 Advanced

- Services and actions
- URDF robot modeling
- TF2 coordinate transforms
- Nav2 navigation stack

**Key Outcome**: Create robot models and navigation systems

## Module 2: Simulation Environments (Weeks 5-7)

### Chapter 5: Robot Simulation with Gazebo

- Gazebo Harmonic setup
- World building with SDF
- ROS-Gazebo bridge
- Sensor and actuator plugins

**Key Outcome**: Simulate robots in physics-based environments

### Chapter 6: Unity for High-Fidelity Simulation

- Unity Robotics Hub
- Photorealistic environments
- Domain randomization
- Synthetic data generation

**Key Outcome**: Create training data with domain randomization

## Module 3: NVIDIA Isaac & AI (Weeks 7-9)

### Chapter 7: NVIDIA Isaac Platform

- Isaac Sim installation and use
- Isaac ROS packages
- Isaac Gym for RL training
- Omniverse Replicator

**Key Outcome**: Train robot policies at scale

### Chapter 8: AI-Powered Perception & Manipulation

- Computer vision pipelines
- Deep learning for perception
- Reinforcement learning for manipulation
- Edge deployment (Jetson)

**Key Outcome**: Implement AI-powered robot behaviors

## Module 4: Humanoid Development (Weeks 9-13)

### Chapter 9: Humanoid Robot Development

- Vision-Language-Action models
- Bipedal locomotion
- Whole-body control
- Human-robot interaction

**Key Outcome**: Build integrated humanoid systems

### Chapter 10: Conversational Robotics & Capstone

- Speech recognition and synthesis
- LLM integration
- Capstone project development
- Presentation and demonstration

**Key Outcome**: Complete autonomous humanoid robot
