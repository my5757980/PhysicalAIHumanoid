# Chapter 8: AI-Powered Perception & Manipulation

**Weeks 8-9 | Module 3: NVIDIA Isaac & AI**

## Learning Objectives

By the end of this chapter, you will be able to:

- Implement computer vision pipelines for robot perception
- Use deep learning models for object detection and segmentation
- Train manipulation policies using reinforcement learning
- Integrate perception with motion planning
- Deploy AI models on edge devices (Jetson)

## Chapter Contents

1. [Computer Vision Basics](./cv-basics) - Image processing for robots
2. [Deep Learning Perception](./deep-learning) - Neural networks for vision
3. [Object Detection & Segmentation](./detection) - Finding objects in scenes
4. [Reinforcement Learning](./rl-manipulation) - Learning to manipulate
5. [Edge Deployment](./edge-deployment) - Running on Jetson

## Key Concepts

| Concept | Description |
|---------|-------------|
| Perception Pipeline | Processing sensor data to understand the environment |
| Object Detection | Locating and classifying objects in images |
| Policy Learning | Training neural networks to control robots |
| Sim-to-Real | Transferring learned behaviors to real robots |

## Prerequisites

- Chapter 7 completed
- Python and PyTorch basics
- Linear algebra and calculus

**20-25 hours** of study and hands-on exercises
