# Quarter Overview

This course spans **13 weeks** (one academic quarter), organized into **4 modules** covering **10 chapters**.

## Course Timeline

```
Week 1-2   │ Chapter 1: Introduction to Physical AI
Week 2-3   │ Chapter 2: Foundations of Humanoid Robotics
Week 3-4   │ Chapter 3: ROS 2 Basics
Week 4-5   │ Chapter 4: ROS 2 Advanced
Week 5-6   │ Chapter 5: Gazebo Simulation
Week 6-7   │ Chapter 6: Unity Simulation
Week 7-8   │ Chapter 7: NVIDIA Isaac Platform
Week 8-9   │ Chapter 8: AI Perception & Manipulation
Week 9-11  │ Chapter 9: Humanoid Development
Week 11-13 │ Chapter 10: Conversational Robotics & Capstone
```

## Module Breakdown

### Module 1: ROS 2 Foundations (Weeks 1-5)

**Chapters 1-4**

Build your foundation in robotics software:

| Week | Topic | Key Skills |
|------|-------|------------|
| 1-2 | Physical AI Introduction | Concepts, environment setup |
| 2-3 | Humanoid Foundations | Kinematics, sensors, actuators |
| 3-4 | ROS 2 Basics | Nodes, topics, publishers/subscribers |
| 4-5 | ROS 2 Advanced | Services, URDF, TF2, Nav2 |

**Milestone**: Working ROS 2 robot with navigation

### Module 2: Simulation Environments (Weeks 5-7)

**Chapters 5-6**

Master simulation tools for robot development:

| Week | Topic | Key Skills |
|------|-------|------------|
| 5-6 | Gazebo Simulation | World building, ROS-Gazebo bridge |
| 6-7 | Unity Simulation | Photorealism, domain randomization |

**Milestone**: Robot operating in custom simulated environments

### Module 3: NVIDIA Isaac & AI (Weeks 7-9)

**Chapters 7-8**

Leverage NVIDIA's robotics ecosystem and AI:

| Week | Topic | Key Skills |
|------|-------|------------|
| 7-8 | NVIDIA Isaac Platform | Isaac Sim, Isaac ROS, Isaac Gym |
| 8-9 | AI Perception & Manipulation | CV, deep learning, RL |

**Milestone**: AI-powered robot with perception and manipulation

### Module 4: Humanoid Development (Weeks 9-13)

**Chapters 9-10**

Build complete humanoid robot systems:

| Week | Topic | Key Skills |
|------|-------|------------|
| 9-11 | Humanoid Development | VLA, locomotion, whole-body control |
| 11-13 | Conversational Robotics & Capstone | Speech, LLM integration, final project |

**Milestone**: Autonomous conversational humanoid robot

## Assessment Structure

| Component | Weight | Description |
|-----------|--------|-------------|
| Weekly Labs | 40% | Hands-on exercises each week |
| Midterm Project | 20% | Module 2 simulation project |
| Final Capstone | 30% | Autonomous humanoid robot |
| Participation | 10% | Discussion and peer review |

## Time Commitment

- **Lectures/Reading**: 3-4 hours/week
- **Labs/Exercises**: 5-6 hours/week
- **Projects**: Additional time during project weeks

**Total**: ~10-12 hours/week

## Prerequisites Check

Before starting, ensure you have:

- [ ] Ubuntu 22.04 installed (or WSL2 on Windows)
- [ ] Python 3.10+ with basic proficiency
- [ ] Git installed and configured
- [ ] NVIDIA GPU (recommended for Module 3+)
- [ ] 16GB RAM minimum
