# Physical AI & Humanoid Robotics

Welcome to the AI-Native Textbook for the **Physical AI & Humanoid Robotics** course. This comprehensive guide will take you from the foundations of embodied intelligence to building fully autonomous humanoid robots.

## What You'll Learn

This textbook covers the complete curriculum for mastering Physical AI and humanoid robotics:

- **ROS 2 Mastery**: Learn the Robot Operating System 2, the middleware that powers modern autonomous systems
- **Simulation Environments**: Build and test robots in Gazebo, Unity, and NVIDIA Isaac Sim before real-world deployment
- **AI Integration**: Connect large language models, computer vision, and reinforcement learning to physical robots
- **Humanoid Development**: Master bipedal locomotion, manipulation, and human-robot interaction

## Course Structure

The course is organized into **10 chapters** across **4 modules**:

| Module | Chapters | Focus Area |
|--------|----------|------------|
| Module 1 | 1-4 | ROS 2 Foundations |
| Module 2 | 5-6 | Simulation (Gazebo & Unity) |
| Module 3 | 7-8 | NVIDIA Isaac & AI Perception |
| Module 4 | 9-10 | Humanoid Development & Capstone |

## Prerequisites

Before starting this course, you should have:

- Basic Python programming experience
- Familiarity with Linux command line
- Understanding of basic mathematics (linear algebra, calculus)
- Access to a computer with Ubuntu 22.04 (or WSL2 on Windows)

## Hardware Requirements

While most exercises can be completed in simulation, the full experience includes:

- **Development Machine**: Ubuntu 22.04, 16GB RAM minimum, NVIDIA GPU recommended
- **Optional Hardware**: Raspberry Pi 5, depth cameras, servo motors for physical builds
