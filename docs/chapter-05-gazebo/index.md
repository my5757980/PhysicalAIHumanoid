# Chapter 5: Robot Simulation with Gazebo

**Weeks 5-6 | Module 2: Simulation Environments**

## Learning Objectives

By the end of this chapter, you will be able to:

- Install and configure Gazebo Harmonic
- Create simulated worlds with obstacles and terrain
- Integrate ROS 2 with Gazebo using ros_gz
- Add sensors and actuators to simulated robots
- Run physics-based simulations for robot testing

## Chapter Contents

1. [Gazebo Installation](./installation) - Setting up Gazebo Harmonic
2. [World Building](./world-building) - Creating simulation environments
3. [ROS-Gazebo Bridge](./ros-gz-bridge) - Connecting ROS 2 and Gazebo
4. [Sensor Simulation](./sensors) - Cameras, LiDAR, IMU
5. [Physics & Dynamics](./physics) - Realistic robot behavior

## Key Concepts

| Concept | Description |
|---------|-------------|
| SDF | Simulation Description Format for worlds and models |
| ros_gz | Bridge between ROS 2 and Gazebo |
| Physics Engine | Simulates forces, collisions, and dynamics |
| Sensor Plugins | Virtual sensors that mimic real hardware |

## Prerequisites

- Chapter 4 completed
- URDF understanding
- ROS 2 pub/sub proficiency

## Estimated Time

**12-15 hours** of study and hands-on exercises
