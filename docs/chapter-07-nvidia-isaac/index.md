# Chapter 7: NVIDIA Isaac Platform

**Weeks 7-8 | Module 3: NVIDIA Isaac & AI**

## Learning Objectives

By the end of this chapter, you will be able to:

- Set up NVIDIA Isaac Sim on your workstation
- Use Isaac ROS for GPU-accelerated perception
- Train robot policies in Isaac Gym
- Deploy Isaac-trained models to real robots
- Leverage NVIDIA's robotics ecosystem effectively
